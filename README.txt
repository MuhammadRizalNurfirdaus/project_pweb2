 Cilengkrang-Web-Wisata
======================

Panduan Instalasi:
1. Import file `database/cilengkrang_db.sql` ke MySQL.
2. Letakkan folder `public/img/` berisi foto-foto di direktori server.
3. Sesuaikan user/password database di `config/config.php`.
4. Akses via browser: http://localhost/Cilengkrang-Web-Wisata/index.php

Fitur:
- Auth: login, register, logout (Admin & User)
- Admin: kelola wisata, booking, galeri foto
- User: lihat wisata, booking, lihat galeri
- Desain responsif

Kontak jika ada kendala: admin@cilengkrang.local
